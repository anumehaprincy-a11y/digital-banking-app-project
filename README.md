# Digital Banking Application Enhancement

## Overview
This project simulates an Agile-based enhancement of a digital banking application, focusing on improving user experience, transaction efficiency, and workflow management.

## Objectives
- Improve user navigation and transaction flow
- Enhance feature usability and performance
- Ensure efficient tracking and delivery of tasks

## Methodology
- Agile (Scrum framework)
- Backlog creation and prioritization
- Sprint-based execution
- Jira-style workflow tracking

## Key Contributions
- Created and managed 10+ user stories aligned with business requirements  
- Maintained a backlog of 20+ tasks with clear prioritization  
- Simulated sprint cycles and tracked progress across stages (To Do, In Progress, Done)  
- Identified risks and dependencies to ensure smooth execution
